Product Recommendation System using Sentiment Analysis on E-Commerce Application

Introduction

The main aim of this paper is to provide a recommender system for customers, by using machine learning algorithms like Naive Bayes and Support Vector Machine based on accuracy, comparing performance metrics.

Methodology

Methodology involves several steps like gathering a valid data set, pre-processing steps to remove unnecessary data, collaborative filtering of text, applying machine learning algorithms.

Results

Experimenting happens in two stages. Experiment 1 outcome is compared with the second one based on the performance metrics. Two machine learning algorithms are used in this technique like Naive Bayes, Support Vector Machine.

Discussion

From the results obtained, Naive Bayes machine has the most precise conclusion. So, using sentiment analysis for recommender system, Naive Bayes has the highest preference over support vector machine.

Conclusion

The thesis recognizes the importance of users having a clear understanding of how recommendations are generated, contributing to a more ethical and user-friendly recommendation system. According to the experimental outcomes, the Support Vector Machine algorithm performs the best, with an accuracy rate of 95.03%.
